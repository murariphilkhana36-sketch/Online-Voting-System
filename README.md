# Online-Voting-System
A secure, user-friendly digital platform that enables registered voters to cast their ballots remotely via the internet.
This is a web-based Online Voting System.
It allows registered users to securely cast votes through an online platform.

Key features include:

User authentication
Secure vote submission
Result calculation

The focus was on making the system user-friendly and secure.

Tech used: HTML, CSS, backend logic (basic).
